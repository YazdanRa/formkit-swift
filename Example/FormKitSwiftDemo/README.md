# FormKitSwiftDemo

`FormKitSwiftDemo` is a small native iOS app for exercising the public `FormKitSwift` package.

## Setup

Generate the Xcode project:

```bash
cd Example/FormKitSwiftDemo
xcodegen generate
```

Then build or run `FormKitSwiftDemo.xcodeproj` on an iOS simulator.

The demo uses only local sample schemas and the local package path dependency. It does not require networking, persistence, OpenAI, or Assist configuration.
